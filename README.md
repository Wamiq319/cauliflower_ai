# Cauliflower AI

## Project Overview
Cauliflower AI is a Django-based web application designed to assist farmers in detecting diseases in cauliflower plants using an AI model. It provides a platform for farmers to upload images of their plants, get disease predictions, and receive suggestions for prevention and treatment. The system also allows doctors to review cases and provide further assistance.

## Features
- **Disease Detection:** Upload plant images for AI-powered disease identification.
- **Confidence Threshold:** Filters out non-cauliflower images or unclear detections.
- **Disease Information:** Access prevention, treatment, and best practice suggestions for various cauliflower diseases.
- **Case Management:** Farmers can open cases for expert review by doctors.
- **User Roles:** Separate dashboards and functionalities for Farmers, Doctors, and Admins.
